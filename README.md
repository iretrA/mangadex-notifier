A notifier for mangadex.

## Work in progress
Will be updated periodically.

# Limitations
It cannot, so far, notify you on your latest updates. This may be added in later updates. This 1.0.0 is essentially a beta version for testing and can only notify for a specific manga. Right now, it only does English, the checking time cannot be changed, and may be off set by the actual update time by a few minutes. This is because the public api is being used and it needs to avoid being rate limited.

# Issues
If you actually use this, please make an issue on this repository if you find any bugs. So far, a log report has not been added yet and you may have to just say what happened before the issue/bug.
